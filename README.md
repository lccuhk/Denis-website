# 🏓 双人乒乓球对战游戏

一个基于 HTML5 Canvas 的经典双人乒乓球对战游戏，具有炫酷的霓虹风格和流畅的游戏体验。

## ✨ 游戏特色

- 🎮 **双人本地对战**：支持两名玩家在同一台电脑上对战
- 🎨 **赛博朋克风格**：霓虹配色，发光特效，视觉效果出众
- ⚡ **真实物理**：球速随击球次数逐渐加快，增加游戏挑战性
- 🎯 **角度控制**：用球拍的不同位置击球可以改变球的反弹角度
- 📊 **实时计分**：清晰的计分板显示双方得分
- ⏸️ **暂停功能**：随时暂停/继续游戏
- 🏆 **获胜判定**：先得 5 分获胜，并有胜利动画效果

## 🎯 操作说明

### 玩家 1（青色球拍）
| 按键 | 功能 |
|------|------|
| `W` | 向上移动 |
| `S` | 向下移动 |

### 玩家 2（紫色球拍）
| 按键 | 功能 |
|------|------|
| `↑` | 向上移动 |
| `↓` | 向下移动 |

### 全局控制
| 按键 | 功能 |
|------|------|
| `空格` | 开始/暂停游戏 |
| `R` | 重新开始 |

## 🚀 运行步骤

### 方法一：直接打开（最简单）
1. 双击 `games/pong_game.html` 文件
2. 用浏览器打开即可游玩

### 方法二：本地服务器运行（推荐）
```bash
# 进入项目目录
cd mini-pong-game

# 启动本地服务器（Python 3）
python3 -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后在浏览器中访问：`http://localhost:8000/games/pong_game.html`

### 方法三：GitHub Pages
1. 开启仓库的 GitHub Pages 功能
2. 访问：`https://<your-username>.github.io/mini-pong-game/games/pong_game.html`

## 🎮 游戏玩法

1. **点击游戏区域**获得键盘焦点
2. 按 **空格键** 开始游戏
3. 两名玩家分别控制各自的球拍
4. 让球越过对方球拍即可得分
5. 先得到 5 分的玩家获胜！

💡 **技巧提示**：用球拍的边缘击球可以让球以更大的角度反弹，增加对手的防守难度！

## 🎯 游戏规则

1. 游戏开始时，球会随机向一方发球
2. 每次击球后，球速会略微增加
3. 球碰到上下边界会反弹
4. 球越过对方底线时，本方得分
5. 先得到 5 分的玩家获胜
6. 获胜后可以按 `R` 键或点击"重新开始"按钮开始新一局

## 📝 注意事项

- 开始游戏前，请确保点击了游戏区域以获得键盘焦点
- 如果按键没有响应，请检查是否有其他程序占用了快捷键
- 建议使用 Chrome、Firefox、Safari 等现代浏览器游玩

### 常见问题及解决方法

#### 1. 游戏画面不显示或黑屏
**可能原因**：浏览器不支持 Canvas API 或禁用了 JavaScript

**解决方法**：
- 确保浏览器已启用 JavaScript
- 尝试按 `F5` 或 `Cmd+R` 刷新页面
- 尝试使用无痕/隐私模式打开
- 清除浏览器缓存后重新打开

#### 2. 按键没有响应
**可能原因**：页面没有获得焦点，或浏览器快捷键冲突

**解决方法**：
- 点击游戏区域确保获得焦点
- 检查是否有浏览器插件占用了快捷键（如 `W`/`S`/`↑`/`↓`）
- 尝试关闭其他可能占用快捷键的程序
- 在 macOS 上，检查系统设置→键盘→快捷键是否有冲突

#### 3. 方向键控制时页面滚动
**可能原因**：浏览器默认行为没有被阻止

**解决方法**：
- 确保点击了游戏区域
- 尝试刷新页面
- 如果问题持续，尝试使用其他浏览器

#### 4. 游戏运行卡顿
**可能原因**：浏览器硬件加速未开启，或后台程序过多

**解决方法**：
- 关闭浏览器中其他不必要的标签页
- 开启浏览器硬件加速：
  - **Chrome**：设置 → 系统 → "使用硬件加速模式"
  - **Firefox**：设置 → 常规 → 性能 → "使用硬件加速"
- 尝试降低浏览器缩放比例（`Ctrl/Cmd + 0` 重置为 100%）
- 更新浏览器到最新版本

#### 5. 空格键无法开始游戏
**可能原因**：浏览器插件或扩展拦截了空格键

**解决方法**：
- 点击游戏区域后再按空格键
- 尝试禁用广告拦截插件或其他浏览器扩展
- 尝试使用 `R` 键先重置游戏，再按空格开始

#### 6. 移动端浏览器问题
**注意**：本游戏为双人本地对战游戏，**不支持触屏操作**，需要物理键盘

**解决方法**：
- 请使用电脑浏览器游玩
- 连接外接键盘到移动设备（不保证完全兼容）

#### 7. 其他问题
如果以上方法都无法解决，请尝试：
1. 更新浏览器到最新版本
2. 尝试使用其他浏览器
3. 检查电脑是否安装了必要的系统更新
4. 在 GitHub 提交 Issue 描述你的问题
